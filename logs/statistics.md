# NumPy & Pandas Statistics

## 학습일

* 2026-07-08

## Goal

* NumPy와 Pandas에서 기본 통계 함수를 사용할 수 있다.
* 평균, 중앙값, 최댓값, 최솟값, 표준편차의 의미를 이해한다.
* 통계값을 단순히 계산하는 것이 아니라 데이터의 특성을 해석하는 방법을 익힌다.

## Implementation

파일: `exercises/statistics.py`

구현 내용

* NumPy 배열의 평균(`mean`), 중앙값(`median`), 최댓값(`max`), 최솟값(`min`), 표준편차(`std`)를 계산했다.
* Pandas DataFrame의 `score` 컬럼에 대해 동일한 통계값을 계산했다.
* 두 개의 반(Class A, Class B)의 평균과 표준편차를 비교하여 데이터 분포를 분석했다.

핵심 학습

* 평균은 데이터의 대표값을 나타낸다.
* 중앙값은 이상치의 영향을 적게 받는 중심값이다.
* 표준편차는 데이터가 평균으로부터 얼마나 퍼져 있는지를 나타낸다.
* NumPy와 Pandas는 대부분 동일한 통계 함수를 제공하지만 호출 방식에는 차이가 있다.

## Learned

* NumPy는 `np.median()`처럼 라이브러리 함수를 사용하는 경우가 있으며, Pandas는 `Series.median()`처럼 객체 메서드를 사용하는 경우가 많다.
* 평균이 같더라도 표준편차가 다르면 데이터의 분포는 크게 달라질 수 있다.
* 통계 함수는 단순한 계산이 아니라 데이터의 특성을 요약하고 해석하는 도구이다.

## 실무 활용

* 데이터 탐색(EDA)의 첫 단계에서 평균, 중앙값, 표준편차를 통해 데이터의 특성을 빠르게 파악한다.
* 표준편차는 점수 분포, 매출 변동성, 센서 데이터의 안정성 등을 분석할 때 자주 활용된다.
* `describe()`를 사용하면 주요 통계 정보를 한 번에 확인할 수 있으며, 데이터 품질을 점검하는 첫 단계로 활용된다.

## NumPy ↔ Pandas 연결점

| NumPy               | Pandas                 |
| ------------------- | ---------------------- |
| `scores.mean()`     | `df["score"].mean()`   |
| `np.median(scores)` | `df["score"].median()` |
| `scores.max()`      | `df["score"].max()`    |
| `scores.min()`      | `df["score"].min()`    |
| `scores.std()`      | `df["score"].std()`    |

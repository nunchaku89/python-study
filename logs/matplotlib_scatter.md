# Visualization - Scatter Plot

## 학습일

* 2026-07-13

## Goal

* Scatter Plot을 이용하여 두 변수의 관계를 시각화한다.
* 양의 상관관계와 음의 상관관계를 구분한다.
* 상관관계와 인과관계의 차이를 이해한다.

## Implementation

파일

* `exercises/matplotlib_scatter.py`

구현 내용

* 공부 시간과 시험 점수를 Scatter Plot으로 표현하였다.
* 운동 시간과 체지방률을 Scatter Plot으로 표현하였다.
* 각 데이터의 상관관계를 해석하였다.

## 핵심 학습

* Scatter Plot은 두 변수 간의 관계를 표현하는 그래프이다.
* 점이 오른쪽 위 방향으로 분포하면 양의 상관관계이다.
* 점이 오른쪽 아래 방향으로 분포하면 음의 상관관계이다.
* Scatter Plot은 변수 간의 관계를 보여주지만 원인을 증명하지는 않는다.

## Learned

* 상관관계는 인과관계를 의미하지 않는다.
* 데이터 분석에서는 관찰된 사실과 추론을 구분해야 한다.
* 숨은 변수(Confounding Variable)가 결과에 영향을 줄 수 있다.

## 프로젝트 적용

현재 Student Score Analysis 데이터에는 관계를 분석할 수 있는 연속형 변수가 부족하다.

향후 Sales Analysis 및 Public Data Analysis 프로젝트에서 Scatter Plot과 상관관계 분석을 적극 활용할 예정이다.

## 실무 활용

Scatter Plot은 다음과 같은 관계를 분석할 때 자주 사용된다.

* 광고비 ↔ 매출
* 공부 시간 ↔ 시험 점수
* 운동 시간 ↔ 체지방률
* 기온 ↔ 전력 사용량

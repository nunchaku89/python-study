## Goal

* `pivot_table()`의 개념과 목적을 이해한다.
* 행(index)과 열(columns)을 기준으로 데이터를 요약하는 방법을 학습한다.
* 다양한 집계 함수(`mean`, `max`, `min`, `count`)를 활용하여 보고서 형태의 데이터를 생성한다.

## Implementation

파일: `exercises/pivot_table.py`

구현 내용

* 전공별 평균 점수를 `pivot_table()`로 계산했다.
* 전공을 행, 등급을 열로 하여 평균 점수를 집계했다.
* 학생 수를 집계했다.
* 평균, 최고점, 최저점을 각각 `pivot_table()`로 출력했다.

## 핵심 학습

* `pivot_table()`은 데이터를 보고서 형태로 재구성하는 기능이다.
* `groupby()`는 그룹을 생성하는 것이 목적이고, `pivot_table()`은 결과를 표 형태로 표현하는 것이 목적이다.
* `values`, `index`, `columns`, `aggfunc`의 역할을 이해했다.
* 하나의 기준뿐 아니라 두 개 이상의 기준으로 데이터를 요약할 수 있다.

## Learned

* `pivot_table()`은 Excel의 피벗 테이블과 동일한 개념이다.
* `groupby()`와 달리 행과 열을 동시에 기준으로 집계할 수 있다.
* 집계 함수만 변경하면 평균, 최대값, 최소값, 개수를 쉽게 계산할 수 있다.
* 분석 결과를 보고서 형태로 표현할 때 매우 유용하다.

## 실무 활용

* 부서별 · 직급별 평균 연봉
* 지역별 · 상품군별 매출

* 전공별 · 학년별 평균 성적

2차원 요약 보고서를 생성할 때 가장 많이 활용된다.

## groupby() ↔ pivot_table()

| groupby()     | pivot_table()      |
| ------------- | ------------------ |
| 데이터를 그룹으로 묶는다 | 데이터를 표 형태로 재구성한다   |
| 계산 중심         | 보고서 중심             |
| 하나의 기준에 적합    | 여러 기준을 동시에 표현하기 적합 |

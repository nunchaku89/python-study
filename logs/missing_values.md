# Missing Values (NaN) Handling

## 학습일

* 2026-07-08

## Goal

* 결측치(Missing Value)의 개념을 이해한다.
* `NaN`의 의미와 Pandas에서의 표현 방식을 익힌다.
* `isna()`, `dropna()`, `fillna()`를 이용하여 결측치를 확인하고 처리하는 방법을 학습한다.
* 데이터 분석에서 결측치 처리의 중요성을 이해한다.

## Implementation

파일: `exercises/missing_values.py`

구현 내용

* 결측치(`NaN`)가 포함된 학생 데이터를 DataFrame으로 생성했다.
* `isna()`를 이용하여 결측치 위치를 확인했다.
* `isna().sum()`을 이용하여 컬럼별 결측치 개수를 확인했다.
* `dropna()`를 이용하여 결측치가 포함된 행을 제거했다.
* 점수(`score`) 컬럼의 평균을 계산하여 `fillna()`로 결측치를 대체했다.
* `major` 컬럼에 결측치를 추가한 후 특정 컬럼의 결측치만 제거하는 방법을 실습했다.

## 핵심 학습

* Pandas는 결측치를 `NaN(Not a Number)`으로 표현한다.
* `isna()`는 결측치 여부를 Boolean 값으로 반환한다.
* `dropna()`는 기본적으로 결측치가 하나라도 있는 행을 제거한다.
* `fillna()`는 결측치를 원하는 값으로 대체하지만 원본 DataFrame은 변경하지 않으므로 결과를 저장하거나 다시 할당해야 한다.
* 특정 컬럼만 기준으로 결측치를 처리할 수 있으며, `dropna(subset=[...])`을 사용하면 더 간결하게 구현할 수 있다.

## Learned

* 실제 데이터는 항상 완전하지 않으며 결측치를 고려한 분석이 필요하다.
* 결측치를 무조건 제거하는 것이 아니라 데이터의 특성에 맞게 제거하거나 대체하는 전략을 선택해야 한다.
* 평균으로 대체하는 방법은 가장 기본적인 결측치 처리 방법 중 하나이다.
* 결측치 처리 이후에 통계 분석이나 머신러닝 모델을 적용하는 것이 일반적인 데이터 분석 흐름이다.

## 실무 활용

* CSV, Excel, 데이터베이스에서 가져온 데이터에는 누락된 값이 자주 존재한다.
* 고객 정보, 센서 데이터, 설문조사, 로그 데이터 등 다양한 데이터에서 결측치 처리는 필수적인 전처리 과정이다.
* 분석 목적에 따라 결측치를 제거(`dropna`)하거나 평균, 중앙값, 최빈값 등으로 대체(`fillna`)하는 전략을 선택한다.

## NumPy ↔ Pandas 연결점

| NumPy        | Pandas                   |
| ------------ | ------------------------ |
| `np.nan`     | `NaN`                    |
| `np.isnan()` | `isna()`                 |
| 배열의 결측치 확인   | DataFrame/Series의 결측치 확인 |
| -            | `dropna()`               |
| -            | `fillna()`               |

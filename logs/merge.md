* `merge()`를 이용하여 데이터 분석에 필요한 정보를 확장하는 방법을 익힌다.

## 핵심 학습

* `merge()`는 공통 Key를 기준으로 여러 DataFrame을 연결한다.
* 새로운 컬럼이 추가되어도 `merge()` 코드는 변경되지 않는다.
* `merge()`는 데이터를 계산하는 기능이 아니라 기존 데이터를 연결하는 기능이다.
* 데이터 분석에서는 하나의 데이터보다 여러 데이터셋을 결합하여 사용하는 경우가 많다.

## Learned

* DataFrame은 독립적인 데이터를 표현한다.
* `merge()`를 이용하면 여러 데이터셋을 하나의 분석용 데이터셋으로 만들 수 있다.
* 계산을 통해 생성하는 데이터(`apply`)와 원래 존재하는 데이터를 결합하는 것(`merge`)은 목적이 다르다.
* 데이터의 관계(Key)를 설계하는 것이 데이터 분석의 중요한 과정이다.

## 실무 활용

* 주문 데이터 + 상품 정보
* 고객 정보 + 구매 내역
* 사원 정보 + 부서 정보
* 공공데이터 여러 파일 결합
* 로그 데이터 + 사용자 정보

관계형 데이터베이스의 JOIN과 동일한 개념으로 활용된다.

## SQL ↔ Pandas 연결

| SQL      | Pandas      |
| -------- | ----------- |
| JOIN     | `merge()`   |
| GROUP BY | `groupby()` |
| WHERE    | Filtering   |
| SELECT   | 컬럼 선택       |

## 프로젝트 개선 사항

## Goal

* `groupby()`의 개념을 이해한다.
* 특정 컬럼을 기준으로 데이터를 그룹화하는 방법을 익힌다.
* 그룹별 평균, 최고값, 최저값, 개수를 계산한다.
* 반복적인 조건문 대신 `groupby()`를 활용하여 코드를 단순화한다.

## Implementation

파일: `exercises/groupby.py`

구현 내용

* 전공별 학생 수를 계산했다.
* 전공별 최고 점수를 계산했다.
* `agg()`를 이용하여 평균, 최고점, 최저점, 학생 수를 한 번에 집계했다.

## 핵심 학습

* `groupby()`는 데이터를 같은 값을 가진 그룹으로 묶는 기능이다.
* 그룹이 만들어진 이후 `mean()`, `max()`, `min()`, `count()` 등의 집계 함수를 적용할 수 있다.
* `agg()`를 사용하면 여러 통계 함수를 한 번에 계산할 수 있다.
* 동일한 조건을 반복해서 작성하는 것보다 `groupby()`를 사용하는 것이 확장성과 유지보수 측면에서 훨씬 유리하다.

## Learned

* `groupby()`는 평균을 구하는 함수가 아니라 **그룹을 생성하는 함수**이다.
* 집계 함수는 생성된 그룹에 대해 계산을 수행한다.
* 반복적으로 조건을 작성하던 코드를 하나의 `groupby()`로 대체할 수 있다.
* 그룹 객체를 재사용하면 코드가 더욱 간결해진다.

## 실무 활용

* 부서별 평균 연봉
* 지역별 매출 합계
* 상품 카테고리별 판매량
* 월별 주문 건수
* 고객 등급별 평균 구매 금액

과 같이 특정 기준으로 데이터를 집계하는 대부분의 업무에 활용된다.

## 프로젝트 개선 사항

기존 코드

```python
cs_average = ...
math_average = ...
physics_average = ...
```

↓

개선 코드

```python
major_summary = (
    df.groupby("major")["score"]
      .agg(["mean", "max", "min", "count"])
)
```

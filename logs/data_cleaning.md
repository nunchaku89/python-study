# Data Cleaning Workflow

## 학습일

* 2026-07-08

## Goal

* 분석 전에 데이터를 정제하는 과정을 이해한다.
* 문자열 공백 제거, 데이터 타입 변환, 중복 제거를 수행할 수 있다.
* 정제 전후의 데이터 상태를 비교하는 방법을 익힌다.

## Implementation

파일: `exercises/data_cleaning.py`

구현 내용

* 문자열 컬럼의 앞뒤 공백을 제거했다.
* `score` 컬럼을 문자열에서 정수형으로 변환했다.
* 중복 데이터를 제거했다.
* `info()`를 이용하여 정제 전후의 데이터 구조를 비교했다.

## 핵심 학습

* `str.strip()`을 이용하여 문자열 공백을 제거할 수 있다.
* `astype()`를 이용하여 데이터 타입을 변환할 수 있다.
* `drop_duplicates()`를 이용하여 중복 데이터를 제거할 수 있다.
* 데이터 정제 후에는 반드시 `info()`와 `describe()`를 통해 결과를 검증해야 한다.

## Learned

* 데이터 분석보다 데이터 정제에 더 많은 시간이 소요될 수 있다.
* 잘못된 데이터 타입은 분석 결과에 영향을 줄 수 있다.
* 중복 데이터는 통계 결과를 왜곡할 수 있다.
* `info()`는 반환값이 아닌 정보를 출력하는 함수이므로 `print(info())` 형태가 아니라 `info()` 자체를 호출해야 한다.

## 실무 활용

데이터 분석 프로젝트에서는 대부분 다음 순서로 전처리를 수행한다.

1. 결측치 처리
2. 문자열 정리
3. 데이터 타입 변환
4. 중복 제거
5. 이상치 확인
6. 분석 수행

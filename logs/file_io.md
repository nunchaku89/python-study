# File IO

## 학습일

- 2026-07-07

## Goal
- 파일을 생성하고 내용을 저장하는 방법을 익힌다.
- 파일의 내용을 읽어오는 방법을 이해한다.
- 파일 모드(r, w, a)의 차이를 이해한다.
- with 문을 사용하는 이유를 이해한다.

## Implementation
파일: exercises/file_io.py

구현 내용
- 사용자에게 이름을 입력받는다..
- study_log.txt 파일에 학습 기록을 저장한다.
- "a" 모드를 사용하여 기존 내용을 유지한 채 새로운 기록을 추가한다.

핵심 학습
- input()으로 입력받은 값을 파일에 저장했다.
- with open(...) 구문을 사용하여 파일을 안전하게 열고 자동으로 닫도록 구현했다.

## Learned
- with 문은 파일을 사용한 후 자동으로 닫아 주므로 직접 close()를 호출할 필요가 없다.
- "w" 모드는 기존 내용을 삭제하고 새로 작성한다.
- "a" 모드는 기존 내용을 유지하면서 뒤에 내용을 추가한다.
